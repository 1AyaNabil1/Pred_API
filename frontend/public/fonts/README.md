# Place your custom font files here

Add your custom display font files to this directory:

- `custom-font.woff2` (preferred, best compression)
- `custom-font.woff` (fallback)

The font is already configured in `src/index.css` with fallback to Google Fonts (Playfair Display).

## Example Fonts to Try

- **Playfair Display** (already included via Google Fonts)
- **Cormorant Garamond**
- **Cinzel**
- **Italiana**

If you don't have custom fonts, the UI will use the Google Fonts fallback automatically.
